Flora et Labora — Logo Files
============================
Place logo files here for project use and client handover.

Expected structure:
  logos/JPG/    — Raster logos for print and social media
  logos/PDF/    — Vector logos for professional print
  logos/PNG/    — Transparent background logos for digital use
  logos/SVG/    — Scalable vector for website and app use

Primary web logo used in flora.html:
  Photos and Videos/logo-flora.png

To use the SVG logo instead of PNG in flora.html:
  Find all instances of: Photos%20and%20Videos/logo-flora.png
  Replace with:          logos/SVG/[exact-filename].svg
  (update src attributes in nav, footer, and hero)

Colors:
  Dark Green background logo: use on dark sections (#1C2E1C, #2D4A2D, #1E1E18)
  Light / Cream logo:         use on light sections (#F7F2EA, #FEFCF8)
  Full color logo:            use for print and social media
