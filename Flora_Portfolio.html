04 — DYNAMIC MENU SYSTEM

A full editable
digital menu.

Four-tab menu system (Caffetteria, Pranzo, Aperitivo, Drinks) powered by Google Sheets. The owner can edit items, prices, and mark products as unavailable daily — changes appear on the website within 2 minutes. No developer needed.

Google Sheets CMS
Menu data lives in a Google Sheet. Owner edits a spreadsheet — the site updates automatically.
Daily Availability
Put "X" in the Available column to hide an item. Remove it to bring it back. Instant.
Static Fallback
If the Sheet is unreachable, the site shows a hardcoded fallback menu. Zero downtime.

08 — UI / UX & TECHNICAL FEATURES

Premium interactions,
zero libraries.

The entire site is built with pure HTML5, CSS3, and vanilla JavaScript. No React, no frameworks, no heavy dependencies. Custom cursor, 3D effects, parallax, scroll animations — all hand-crafted for maximum performance and zero bloat.

Custom Cursor
Dot + ring with lerp follow, enlarges on interactive elements. mix-blend-mode: exclusion. Desktop only via media query.
Scroll Animations
IntersectionObserver-based reveal system. 4 variants: fade-up, left, right, scale. 7 stagger delays. One-shot with unobserve.
Frosted Glass Nav
Transparent on load, transitions to backdrop-filter: blur(24px) with saturate(1.4) on scroll >80px. 0.5s smooth easing.
Language Toggle
Full IT/EN bilingual system via data-lang attributes. Single toggle button swaps all content instantly. CSS-driven visibility.
Reservation Modal
Overlay modal with Coworking/Workshop choice cards. CSS scale + translateY entrance. Closes on backdrop click or Escape.
Mobile-First
3 breakpoints (1024/768/480px). Full-screen mobile nav overlay with staggered link animations. All sections reflow cleanly.
SEO Baseline
JSON-LD LocalBusiness, Open Graph meta, semantic HTML5, Italian alt text, FAQ section for AI search signals.
Performance
will-change + translateZ(0) GPU compositing. Lazy loading images. RAF-throttled scroll. prefers-reduced-motion support.

TECH STACK
